# go-srv-discover


```
$ gsr --help
  -redis
        use redis to save result. (default false)
  -socket-redis
        use unix domain socket to connect with redis. (default false)
  -srv string
        string of SRV RR (default "_http._tcp.mxtoolbox.com")
  -ttl int
        TTL of redis-cache (default 20)
```
